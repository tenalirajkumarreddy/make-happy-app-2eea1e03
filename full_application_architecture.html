📊 Core Business Flows
💡 Key Principle: All business logic is server-driven. Frontend triggers actions, backend validates and executes. Offline queue holds actions until sync. Realtime broadcasts changes back to all clients.

🔐 Security & Compliance

✅ Security Measures

  • RLS Policies on 69+ tables
  • JWT Authentication (Supabase & Firebase)
  • KYC Documents in private storage
  • Encryption at rest & in transit
  • Credential Rotation required
  • .env Secrets isolated
  • Audit Logging all changes
  • Rate Limiting on API endpoints
  • CORS Configured properly
  • Input Validation server-side
  • SQL Injection Prevention via ORM
  • XSS Protection via React
  • CSRF Tokens on forms
  • 2FA Support available
  • Session Timeout configured
🛡️ Data Protection: Customer data protected by role-based RLS. KYC documents encrypted and isolated. GPS data logged with timestamp for audit trail. All transactions logged to audit_log table. Staff invitations cryptographically signed.
